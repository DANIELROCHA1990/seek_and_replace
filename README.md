# README

## Este App Rails baseia-se que o usuario pesquisará uma expressão em um arquivo pdf, se essa expressão existir o usuario podera substitui-la, ao substituir será criado um novo arquivo pdf para download

## Ruby version = 3.0.2

## gems utilizadas:
  - gem 'pdf-reader'
  - gem 'prawn'
  - gem 'securerandom'
  - gem 'byebug'
